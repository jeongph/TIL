# CLAUDE.md

이 파일은 Claude Code (claude.ai/code)가 이 저장소에서 작업할 때 참고하는 가이드입니다.

## 저장소 개요

이 저장소는 TIL(Today I Learned) 문서 저장소입니다. 코드 프로젝트가 아닌 학습 기록용 마크다운 문서 모음입니다.

## 디렉토리 구조

- `/docs/TIL/` - 날짜별 학습 기록 (Today I Learned)
- `/docs/etc/` - 기타 메모 및 기록
- `/docs/knowledge/` - 학습한 지식 정리
- `/docs/sequential/` - 순서대로 정리된 가이드
- `/docs/sweat/` - 삽질 기록 (시행착오를 통해 배운 것들)
- `/docs/learning/` - 책/강의 학습 내용

## 콘텐츠 가이드라인

- TIL은 "이쁜 정리"보다 "빠르게 기록"하는 것을 우선
- 파일명 형식: 날짜+주제 (예: `2020-01-16_EC2_setting.md`)

## GitHub 템플릿

- Issue 템플릿: Question, Todo 라벨 사용
- PR 템플릿: 제목, 내용, 코드 특이점 섹션 포함
