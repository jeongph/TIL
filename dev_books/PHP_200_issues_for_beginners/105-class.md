# 클래스란?
자바에서 미친듯이 했기 때문에, 대략적인 개념과 객체지향에 대해 알고있다. 단, 왜 PHP에 객체지향이 필요한지는 의문.

## 클래스 사용 방법
> class 클래스명
{
 클래스 기능
}

## 접근 제한자
기존의 자바와 동일하다
- public
- protected
- private

## 메소드 만들기
> calss test
{
  function 메소드명(){}
}

이것 역시 동일하네.

## 생성자
골때린건 진성으로 하는건지 생성자도 있다.
> = new className();

호출은 다른 언어와 비슷한데, **->** 로 호출한다.
> ex) test -> 함수(매개변수);
